Basic Twitch Mod Commands

/timeout [username] [time]
Temporarily silence a user (1s-2w)
Formats: 30s, 5m, 1h, 1d, 2w (max 2 weeks)
/ban [username]
Permanently ban a user from chat
/warn [username] [reason]
Warn a user in chat
/emoteonly
Enable emoteonly mode
/clear
Clear the entire chat
